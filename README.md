Site feito com HTML e CSS para uma estudante de psicologia.

O site possui 4 telas, cada qual personalizada para seu uso. 
Todos possuem fotos. Em uma das telas, possui um carrosel de fotos com as palestras feitas pela estudante.

O site foi montado com uma base dos cursos da Alura e conhecimento prévio, além de melhorias que foram sugeridas por IA. 

Abra o site pelo link no Vercel: https://andressa-psicologa.vercel.app/
